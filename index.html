Project Overview

This project analyzes 100,000+ NYC yellow taxi trips to identify demand patterns, fare trends, and operational efficiencies. Using Python's data science stack, I processed raw trip data, created interactive visualizations, and applied clustering techniques to locate high-demand zones.


The analysis reveals optimal operating hours, payment preferences, and geographic hotspots that can help taxi companies optimize fleet deployment and pricing strategies. This end-to-end project demonstrates my skills in data cleaning, exploratory analysis, and business intelligence.
